## 🎯 这个理论解决什么问题？

> 为什么玩家会记住"打了一晚上都没出货"的痛苦夜晚，却记不住"每天都挺开心地刷了会儿"的一个月？
>
> 又为什么《艾尔登法环》玩家对"打了四个小时终于击败玛莲尼亚"的记忆是"太爽了"，而不是"前三小时五十九分钟都在受苦"？
>
> 答案反直觉：**人类对一段体验的评价，不是取平均值，而是由两个瞬间决定——最强烈的情感峰值（Peak）和结束时的感受（End）。** 中间有多长、平均多平淡，大脑根本不在乎。

## 📐 核心理论

**峰终定律**（Peak-End Rule）由诺贝尔经济学奖得主 Daniel Kahneman 与其合作者 Barbara Fredrickson 于 1993 年提出。他们在一系列实验中发现：

> 人类在回顾一段体验时，并不会"回放"整个过程的每一刻并取平均值。相反，大脑使用一种**启发式捷径**——只提取两个关键节点的情感强度，加权得出整体评价。

**两个关键节点：**

1. **峰值瞬间（Peak）**：体验过程中**情感强度最高**的时刻（无论是正向还是负向）。注意：痛苦体验的峰值是"最痛的那一刻"，快乐体验的峰值是"最快乐的那一刻"。
2. **终值瞬间（End）**：体验**结束时**的情感强度。

公式化的表达：

```
整体评价 ≈ (Peak 强度 + End 强度) / 2

     ↑ 情感强度
     │
     │        ★ Peak（峰值：击败 Boss 的狂喜）
     │       ╱ ╲
     │      ╱   ╲    ★ End（终值：掉落稀有装备）
     │     ╱     ╲  ╱
     │    ╱       ╲╱
     │   ╱
     │───┴────────────────────→ 时间
     │  （这段"平淡刷怪"的大脑忽略不计）
     │
     │  大脑记忆的整体评价 = (Peak + End) / 2 = 很棒！
```

经典实验（Kahneman, Fredrickson, Schreiber & Redelmeier, 1993）：
- 被试将手浸入 **60秒 14°C 冷水**（条件 A，痛苦体验）
- 另一组将手浸入 **60秒 14°C 冷水 + 额外30秒水温升至 15°C**（条件 B，更长时间但结尾"略微没那么痛"）
- 结果：被试**显著更偏好条件 B**——尽管客观上总痛苦更多（90秒 vs 60秒），但因为结尾体验更好，大脑给出了更高的整体评价。

> 关键洞察：**玩家"记住"的游戏体验 ≠ 玩家"经历"的游戏体验。** 设计师如果只优化"平均每分钟的乐趣"，而忽略了峰值和结尾的设计，就会投入大量精力却换来"玩家觉得一般"的评价。

## ⚙️ 底层机制

把峰终定律想象成**电影的"高光剪辑"**：

- 你看完一部两小时的电影，向朋友推荐时不会说"第3分钟到第7分钟镜头很稳"——你会说"结尾那段太震撼了"或者"中间那个反转我鸡皮疙瘩都起来了"
- 你的大脑就像一个懒惰的剪辑师，从两小时的素材中只剪出**最炸裂的一个画面**和**最后一个画面**，拼在一起作为"整部电影的总结"

在神经科学层面，这个"懒惰剪辑"有三个底层原因：

### 1. 情感记忆的"优势编码"

杏仁核在**情感强度峰值**时会释放大量去甲肾上腺素，这种神经递质充当"记忆胶水"——情感越强，记忆"粘得越牢"。平淡时刻没有足够的神经递质支持，在海马体中只能形成**弱痕迹**，很快被遗忘。

```
情感强度低 → 杏仁核沉默 → 去甲肾上腺素微量 → 海马体弱编码 → 记忆模糊
情感强度高 → 杏仁核激活 → 去甲肾上腺素大量 → 海马体强编码 → 记忆鲜活
```

### 2. "叙事自我" vs "体验自我"

Kahneman 将自我分为两个：

| | 体验自我 (Experiencing Self) | 叙事自我 (Narrating Self) |
|---|---|---|
| **活在** | 当下每一刻 | 事后回顾 |
| **关心** | 此刻感受如何 | "整体怎么样"的故事 |
| **记忆容量** | 约数十秒工作记忆 | 长期记忆 |
| **决策影响力** | 很小（只在当下） | **极大（决定是否再来一次）** |

峰终定律本质上是**叙事自我的工作方式**——它没有完整录像，只有"高光集锦"。决定你是否再打开游戏的，是叙事自我，不是体验自我。

### 3. 结尾的"近因效应"

结尾之所以权重高，是因为**最后几秒的信息在短期记忆中最鲜活**。当你在游戏会话结束后立刻被问到"好玩吗？"，大脑首先提取的就是刚刚发生的最后一幕。这个"近因偏差"使得 End 的权重几乎与 Peak 等量。

## 🎮 游戏设计应用

### 踩坑故事：当每日活动的结尾是"失望"

> 某热门二次元手游的"圣遗物副本"长期被玩家诟病。玩家每天消耗树脂刷副本，**95%的掉落是无用的低属性圣遗物**。
>
> 从峰终定律角度分析这段体验：
> - **峰值**：偶尔出货的狂喜（罕见但极强烈）或连续不出货的暴躁（常见且强烈负向）
> - **终值**：最后一次刷完——体力耗尽，看了一眼背包全是垃圾——关闭游戏
>
> 对于大多数非运气极佳的玩家，Peak 是负面的（"又是垃圾！"的挫败），End 也是负面的（"白花了二十分钟"的空虚）。**整体记忆评价 = 负面 + 负面 = "这游戏真恶心"。**
>
> 而隔壁《崩坏：星穹铁道》在同期的"遗器"系统做了同样的设计，但增加了一个"末尾保底"机制：**每次副本结束时，至少获得一个可用级别的过渡装备**。这个改动没有改变整体的低概率，但它改变了 **End**——玩家关闭游戏时的最后感受从"全是垃圾"变成了"至少有个能用的"。社区口碑差异显著。

这个案例揭示了一个峰终定律在设计中的核心应用：**如果你无法控制 Peak，至少控制 End。**

### 设计手段

| 设计手段 | 峰终机制 | 游戏案例 | 效果 |
|---|---|---|---|
| **Boss战高潮结尾** | 让会话在最兴奋的时刻结束 | 《艾尔登法环》击败 Boss 后的"GREAT ENEMY FELLED"画面+专属音乐 | End 极度正向，玩家带着巨大的满足感放下手柄 |
| **每日签到末位奖励** | 在每日活跃路径的终点放最好的奖励 | 《原神》每日委托最后一弹原石奖励 | End 正向，即使过程中有无聊跑腿 |
| **抽卡动画高潮** | 将不确定性推向情绪最高点再揭示结果 | 《原神》抽卡紫光→金光的升级动画 | Peak 极强（无论结果好坏，情绪都到了顶点） |
| **赛季末段位结算** | 赛季结束时的奖励和荣誉展示 | 《英雄联盟》赛季结算皮肤+段位框 | End 正向，让整个赛季的回忆被"结算荣耀"定调 |
| **副本结算页面优化** | 在每次活动的最后呈现有意义的总结 | 《命运2》副本结算的装备展示和队友感谢 | End 正向，避免"打完了然后呢"的空虚感 |
| **负向 Peak 的补偿机制** | 在挫败时刻后紧接一个小正反馈 | 《只狼》死亡后原地复活不丢失进度 | 降低负向 Peak 的强度，防止整体评价为负面 |
| **意外惊喜结尾** | 在玩家以为结束时给一个彩蛋 | 《英雄联盟》赛季结算后发现邮箱还有隐藏奖励 | End 超预期正向，叙事自我的"故事"变得精彩 |

### 反面案例：中断在低谷

> 某MMO手游设计了"副本限时30分钟"机制。如果超时，所有掉落归零。大量玩家的体验路径是：开心打副本→时间快到了→焦急加速→超时→**什么都没得到→关闭游戏**。
>
> Peak 可能是正向的（中间打得挺爽），但 End 是**灾难性的负向**（白费半小时）。从峰终定律看，这些玩家对游戏的评价主要由这个"白费时间"的终值主导——即使他们在过程中有不错的体验。
>
> 修复方案极简单：超时后仍给予**50%的基础奖励**（而非归零），把 End 从"绝对损失"变成"打折收获"，整体评价立即改善。

## ✅ 设计检查清单

| ✅ 正确做法 | ❌ 错误做法 |
|---|---|
| 每次会话的结尾安排一个正向时刻（哪怕很小） | 会话在"体力耗尽、无事可做"的空虚中结束 |
| 在体验中制造至少一个"值得记住"的峰值时刻 | 整个会话平淡如水，没有高潮也没有低谷 |
| 负向峰值后紧跟一个补偿机制（安慰奖励、复活机会） | 挫败后直接结束会话，让玩家"带着怒气睡觉" |
| Boss 战的击杀瞬间用画面+音效+文字强化 | Boss 倒下后直接切黑屏加载——浪费了最好的 End |
| 抽卡结果展示用渐进式悬念推向情绪顶点 | 抽卡结果瞬间弹出，没有情绪铺垫 |
| 副本结算页面展示成就和数据，让玩家"有东西可看" | 副本结束后直接踢回主城，没有"句号感" |

## 📊 数据验证方法

| 指标 | 含义 | 验证方法 |
|---|---|---|
| **会话终值满意度** | 玩家在结束会话时的情感状态 | 在玩家退出游戏后推送一条简短调查（"刚才的体验如何？1-5分"），与该次会话的 End 时刻事件做关联分析 |
| **峰值时刻识别** | 识别玩家体验中的情感最高点 | 通过行为代理指标推断：操作频率突变、聊天关键词情感分析、 involuntary 停顿（呆住几秒） |
| **回归率与会话终值的关系** | 终值越好，下次回来越快 | 对比"自然退出"vs"体力耗尽退出"vs"团灭退出"的玩家，追踪其下次登录间隔 |
| **NPS 与峰终评分的相关性** | 净推荐值是否与峰终评分一致 | 用峰终公式预测的评分 vs 玩家实际 NPS 评分，相关性应 > 0.6 |

> 关键分析：做一个简单的 A/B 测试。对同一副本，A 组在结束时给一个"额外小奖励"（优化 End），B 组不给。对比两组的"下次副本参与率"。如果 A 组显著更高，峰终效应得到验证。

## 📖 新概念解析

| 术语 | 一句话定义 |
|---|---|
| 峰终定律 (Peak-End Rule) | 人对体验的整体评价主要由情感峰值和结束时的感受决定，而非平均值 |
| 峰值瞬间 (Peak Moment) | 体验过程中情感强度最高的时刻，无论正向还是负向 |
| 终值瞬间 (End Moment) | 体验结束时的情感状态，因近因效应而获得超高权重 |
| 叙事自我 (Narrating Self) | 事后回顾并"讲故事"的自我，依赖峰终记忆，决定未来行为 |
| 体验自我 (Experiencing Self) | 活在当下每一刻的自我，体验丰富但决策权很小 |
| 情感优势编码 (Emotional Superiority Encoding) | 高情感时刻获得更强的记忆编码，因为杏仁核释放去甲肾上腺素 |
| 持续时间忽视 (Duration Neglect) | 人对体验长短不敏感——4小时的快乐和1小时的快乐，如果 Peak 和 End 相同，评价几乎一样 |

- [心流理论](/user-insight-kb/card/flow) — 心流状态是制造正向 Peak 的最佳途径——当玩家进入心流时，情感强度自然达到峰值
- [奖励预测误差](/user-insight-kb/card/reward-prediction-error-games) — 正向预测误差（意外惊喜）是制造 Peak 的核心机制，而多巴胺的响应模式恰好契合峰终编码的需求
- [蔡加尼克效应](/user-insight-kb/card/zeigarnik-effect) — 蔡加尼克效应管"玩家为什么会回来"，峰终定律管"玩家对上次的记忆是好是坏"——前者管回归驱动，后者管回归后的预期质量
- [Hook 上瘾模型](/user-insight-kb/card/hook-model) — Hook 模型的"奖励"环节是 Peak 的主要来源，"投入"环节是 End 的设计窗口

## 💡 个人洞察

峰终定律可能是**被游戏设计师低估得最严重**的行为心理学发现。

行业里大量的设计精力花在"优化平均体验"上——调整数值平衡、压缩匹配时间、减少加载等待、提高画面帧率。这些都没错，但如果从峰终定律的视角审视，这些优化大多在改善"平淡的中间段"——而大脑恰恰不关心中间段。

**真正决定玩家口碑的，是那一个 Peak 和那一个 End。**

举个极端例子：《艾尔登法环》的平均体验包含了大量"被小怪打死""迷路""反复跑尸"的负面时刻。从"每分钟乐趣平均值"看，它远低于很多手游。但它的 Peak（击败 Boss 的瞬间）和 End（通关后对整个旅程的回顾）都极其强烈——所以玩家口碑极高。**叙事自我记住的是"我在交界地冒险的史诗故事"，而不是"我在桌崖地洞迷路40分钟的烦躁"。**

我给设计师的实操建议是：**在每次会话的"最后30秒"上投入不成比例的设计资源。** 这30秒可能是：
- 副本结算页面的动画和音效
- 每日最后一弹委托的额外特效
- 退出游戏前的"今日收获"总结
- 抽卡结束时的结果展示

这30秒的投资回报率，可能比你花三周调整副本数值平衡的回报率还高。

另一个洞察：**负向 Peak 不可怕，可怕的是没有正向 Peak 来覆盖它。** 《黑暗之魂》系列充满了负向 Peak（被 Boss 一击秒杀、掉落悬崖），但它同时提供了极强的正向 Peak（终于击败 Boss），而且正向 Peak 的情感强度压过了负向——所以整体评价是"受苦但值得"。如果只有负向 Peak 没有正向 Peak（比如纯肝的游戏），叙事自我就会写下"这是一段痛苦的回忆"。

## 🔭 下次玩游戏时观察

选一个你最近在玩的游戏，做这个"峰终审计"：

1. **识别你的 Peak**：回想上一次游戏会话，情感最强烈的一刻是什么？是正向的还是负向的？用 1-10 评分。如果是负向的（比如"又被挂机队友坑了"），有没有紧随其后的补偿机制？
2. **识别你的 End**：上次你是怎么结束游戏会话的？最后30秒在做什么？最后的心情是什么？用 1-10 评分。如果 End 总是负面的（体力耗尽、输了比赛、掉线），那这个游戏的"回归预期质量"正在被持续拉低。
3. **对比平均体验**：如果把上次会话的"每分钟乐趣"取平均值，和 Peak-End 的加权评分相比，哪个更接近你对"这游戏好玩吗"的直觉判断？大概率是后者。
4. **做一个实验**：下次玩游戏时，在"最爽的那一刻"主动停下。对比"在峰值结束"和"在无聊处自然结束"的体验，哪种让你更想"马上再开一局"？
